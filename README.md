# Student Management System

A responsive Student Management System developed using PHP, MySQL, HTML, CSS, and Bootstrap.

## Features

- Secure Admin Login
- Add Student
- View Students
- Update Student Information
- Delete Student
- Search Students
- Session-Based Authentication
- Responsive User Interface

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5

## Installation

1. Clone the repository.

```
git clone https://github.com/gklcode/student_management_system.git
```

2. Move the project into your XAMPP `htdocs` folder.

3. Start Apache and MySQL.

4. Create a MySQL database.

5. Import the SQL file.

6. Update the database credentials inside `db.php`.

7. Open:

```
http://localhost/student_management_system/
```
